My Finance Budget Tracker

A Progressive Web Application that Tracks Your Spending

Deployed Site: https://agile-gorge-53171.herokuapp.com/

Github Repo: https://github.com/felicitybrammer/my-finance-budget-tracker





Description

A budget tracking application that allows users to add expenses and deposits to their budget with or without a connection. If the user enters transactions offline, the total will be updated when they're brought back online.

Dependencies: Mongoose, Express
