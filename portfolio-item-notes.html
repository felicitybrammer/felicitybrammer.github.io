Never Forget! Note Taker

A Twitter-style app for sharing your opinions

Deployed Site: https://never-forget-note-taker.herokuapp.com/

Github Repo: https://github.com/felicitybrammer/never-forget-note-taker





Description

An application that allows users to create, save and delete persistent notes. The app uses an Express back-end, and saves and retrieves notes from a JSON file


Built with: JavaScript, Node.js, Express
